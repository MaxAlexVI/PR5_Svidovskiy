## Ответы на вопросы
### Что такое пул соединений *sql.DB и зачем его настраивать?

Пул соединений - набор установленных соединений с БД, которые переиспользуются между запросами. Без использования пула каждый запрос будет создавать нового соединение и нагружать БД
### Почему используем плейсхолдеры $1, $2?

Плейсхолдеры используются для защиты от SQL-инъекций
### Чем Query, QueryRow и Exec отличаются?

Query используется для SELECT'а с большим количество строк (когда нам нужно вывести много строк, например лист всех тасков либо выполненные таски)
QueryRow используется для SELECT'а с одной строкой (когда нам нужно вывести одну конкретную строку, например, таск по id)
Exec используется для модификации данных и не возвращают строк(INSERT/UPDATE/DELETE).

### Обоснование транзакций и настроек пула

Транзакции используются для создания группы запросов, которые выполняется как единое целое. Выполняются либо все успешно, либо ни одна.
